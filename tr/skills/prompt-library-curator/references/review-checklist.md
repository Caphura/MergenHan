# Review Checklist

Bu checklist, `prompt-library-curator` skill'i ile prompt library gozden gecirilirken kullanilabilir.

## Yapilandirma

- Icerik dogru klasorde mi?
- Dosya adi `kebab-case` mi?
- Gerekliyse modul ayristirmasi yapilmis mi?

## Metadata

- `id` deseni dogru mu?
- `status` izinli degerlerden biri mi?
- `depends_on` gercek bagimliliklari yansitiyor mu?
- `last_reviewed` guncel mi?

## Paketleme

- Blueprint ile paketlenmis skill arasindaki bag korunuyor mu?
- `SKILL.md` sade ve tetikleyici acik mi?
- Yonetisim verileri `meta.yaml` icinde mi?

## Bulunabilirlik

- README baglantilari guncel mi?
- Katalog girdileri yeni dosyalari gosteriyor mu?
- Ornek kompozisyonlar hala gecerli mi?
