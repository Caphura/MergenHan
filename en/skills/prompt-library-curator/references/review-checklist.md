# Review Checklist

Bu checklist, `prompt-library-curator` skill'i ile prompt library gozden gecirilirken kullanilabilir.

## Yaplistingdirma

- Icerik correct klasorde mi?
- Dosya adi `kebab-case` mi?
- Gerekliyse modul separatemasi yapilmis mi?

## Metadata

- `id` deseni correct mu?
- `status` izinli degerlerden biri mi?
- `depends_on` gercek dependenciesi yansitiyor mu?
- `last_reviewed` current mi?

## Paketleme

- Blueprint ile paketlenmis skill arasindaki bag korunuyor mu?
- `SKILL.md` sade ve tetikleyici clear mi?
- Yonetisim verileri `meta.yaml` icinde mi?

## Bulunabilirlik

- README links current mi?
- Katalog girdileri yeni dosyalari gosteriyor mu?
- Example kompozisyonlar hala gecerli mi?
